## 4 Rules of Simple Design

Consists of:

1. Tests Pass
2. Expresses Intent
3. No Duplication
4. Small

Further reading - [Corey Haines: Understanding the Four Rules of Simple Design](https://leanpub.com/4rulesofsimpledesign).
